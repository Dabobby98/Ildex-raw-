# im Event - Event Landing Page

## Overview
A beautiful, responsive HTML/CSS/JavaScript event landing page template with PHP backend support for contact forms. This project showcases a Christmas Market event with sections for schedule, sponsors, pricing, and location.

## Project Structure
- **Frontend**: Static HTML5 pages with Bootstrap, jQuery, and various UI plugins
- **Backend**: PHP for contact form handling
- **Assets**: 
  - CSS themes (multiple color schemes available)
  - JavaScript libraries (jQuery, Bootstrap, Owl Carousel, etc.)
  - Images and icons
  - PHP mail handlers

## Technologies Used
- HTML5/CSS3
- JavaScript (jQuery 2.1.1)
- Bootstrap 3
- Font Awesome
- Owl Carousel 2
- PHP 8.2 (for form handling)
- Various jQuery plugins (countdown, prettyPhoto, superfish, waypoints)

## Key Features
- Responsive design
- Multiple theme color options
- Event countdown timer
- Image carousel/slider
- Contact and registration forms
- Google Maps integration
- Sponsor showcase
- Event schedule display

## Pages
- `index.html` - Main landing page
- `coming-soon-1.html`, `coming-soon-2.html`, `coming-soon-3.html` - Coming soon variants
- `register.html` - Event registration page
- `error-404.html` - 404 error page
- `error-page.html` - General error page

The project runs using PHP's built-in development server on port 5000.
